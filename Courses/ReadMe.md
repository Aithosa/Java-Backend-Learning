# Java 后端课程学习计划

## Java部分

Java基本语法暂时没有专门观看课程的计划，等其他部分看的差不多了(应用能力)，在考虑钻研Java语法。

## 课程部分

阅读的书籍参考Code Sheep推荐，书籍部分属于长线学习，其中不乏很多经典书籍，偏向于理论，为面试提供理论基础。而课程属于紧急学习，提升短期内的开发上手能力。

目前后端开发用到的包括**框架**部分和**数据库**部分。
* Spring Boot
* Mybatis
* MongoDB
* MySQL
* Redis
